# herdrdot

Herdr local configuration.

## Keybindings

Prefix is `ctrl+a`.
Custom command descriptions are shown in `ctrl+a` then `?`.

| Key | Action |
| --- | --- |
| `ctrl+w` | Open the workspace navigation picker. |
| `ctrl+n` | Toggle zoom for the current pane. |
| `ctrl+a` then `b` | Toggle the navigation sidebar. |
| `cmd+left/right/up/down` | Move between panes in the current workspace. |
| `cmd+ctrl+left/right` | Move between tabs in the current workspace. |
| `cmd+ctrl+up/down` | Move between workspaces. |
| `ctrl+a` then `ctrl+g` | Open `lazygit` in a temporary pane for the current workspace. |
| `ctrl+a` then `ctrl+e` | Open `nvim` in a temporary pane for the current workspace. |
