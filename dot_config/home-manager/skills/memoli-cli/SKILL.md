# Memoli CLI

Use `memolix` for memoli work. Do not use `memoli serve`.

## Commands

- Help: `memolix help`
- Daily memo: `memolix daily`
- Today's memo: `memolix today`
- Open memo: `memolix memo <name>`
- Tasks: `memolix task <subcommand>`

## Workflow

1. Use normal memoli CLI subcommands.
2. Do not start the MCP server.
3. Keep file edits scoped to the requested memo or task.
